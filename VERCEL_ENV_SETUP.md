# Vercel Environment Variables Setup

Add these two variables to your Vercel project dashboard:

1. `STRIPE_SECRET_KEY` = [Your Stripe Secret Test Key]
2. `STRIPE_PUBLIC_KEY` = [Your Stripe Public Test Key]

## How to Configure in Vercel Dashboard:

1. Go to https://vercel.com/dashboard
2. Select "webpro-local" project
3. Click "Settings" → "Environment Variables"
4. Add each key with appropriate values
5. Save and redeploy

Vercel will automatically restart with new environment variables loaded.
